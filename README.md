# Severance Backend

Node.js + Express + MongoDB backend for shoe store app.

## Features
- JWT Authentication
- CRUD APIs
- Image Upload (Multer)
- MongoDB Atlas

## APIs
- POST /auth/register
- POST /auth/login
- GET /shoes
- POST /shoes
- PUT /shoes/:id
- DELETE /shoes/:id

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
